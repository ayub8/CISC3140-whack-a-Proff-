Software:
The main game screen has a stop watch on top to time the players.
The main screen has a main playing area in the middle of the screen where objects move around and the game is played.
The main game area is landscape on desktop and tablet and vertical on mobile devices.
On desktop and tablet, the cursor should turn into a hammer, which will be animated to slam on the moving objects
At the beginning of the game, there is a start button that will strat the stop watch.
Each moving object, if hit, should disappear from the game using an animation.
There are a limited number of moving objects (between 5 and 10) that will appear and disappear on the main area of the game.
At the end of the game, a leaderboard shows up with the highest scores (shortest time to finish the game) on top.
